    About CodeForge
    
    
    
      CodeForge is an AI-powered code refactoring system that uses Retrieval-Augmented Generation (RAG) to analyze, improve, and explain code. It combines large language models with a curated knowledge base of best practices.
    
    

    
    
      
    🧠 What is RAG?
    
      
    
        RAG (Retrieval-Augmented Generation) enhances AI responses by first retrieving relevant context from a knowledge base — in this case, coding best practices, design patterns, and documentation — before generating the improved code. This makes results more accurate and grounded than pure LLM generation.
      
    
    
    

    
    
      
    ⚙ How the System Works
    
      
    
        
    
          
    
            
    User Inputs Code
    
            
    Paste code or upload a file via the frontend interface
    
          
    
        
    
        
    
          
    
            
    RAG Retrieval
    
            
    Backend searches vector DB for relevant best practice documents
    
          
    
        
    
        
    
          
    
            
    LLM Refactoring
    
            
    AI generates improved code with retrieved context as guidance
    
          
    
        
    
        
    
          
    
            
    Result Displayed
    
            
    Issues, refactored code, and explanation are shown side-by-side
